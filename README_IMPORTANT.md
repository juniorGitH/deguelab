## 🌟 QU'EST-CE QUE DEGUE LAB?

**Degue Lab** est une boutique en ligne moderne pour la vente de:
- 🥛 Degue classique et premium
- 🍓 Mélanges avec fruits exotiques
- 💪 Produits protéinés pour sportifs
- 🍨 Yaourts artisanaux

✅ Pas de compte nécessaire
