## 🚀 Pour démarrer

### Méthode simple (Double-clic):
1. Double-cliquez sur `INSTALL.bat`
2. Attendez la fin de l'installation
3. Double-cliquez sur `START.bat`
4. Ouvrez http://localhost:1234

### Méthode ligne de commande:
```bash
npm install
npm start
```
