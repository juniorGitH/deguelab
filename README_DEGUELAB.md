# 🧪 DEGUE LAB

> La révolution du degue artisanal rencontre l'innovation nutritionnelle

## 🌟 À propos

Degue Lab est une boutique en ligne moderne et tendance spécialisée dans la vente de **degue** et **yaourt artisanaux** premium. Nous proposons des créations uniques qui allient tradition et innovation.

### Nos Produits

- 🥛 **Degue Classique** - Authentique et traditionnel
- 🍓 **Degue Fruits Mix** - Mélanges avec fruits exotiques (mangue, ananas, fraise, passion, etc.)
- 💪 **Degue Protein** - Enrichis en protéines pour sportifs et bodybuilders (25-35g de protéines)
- 🍨 **Yaourts** - Nature, aux fruits, et versions protéinées

## 💻 Installation

```bash
# Installer les dépendances
npm install

# Lancer en développement
npm start

# Build pour production
npm run build
```
