## 🔧 DÉPANNAGE

### Le site ne démarre pas?
1. Vérifiez que Node.js est installé: `node --version`
2. Supprimez `node_modules` et `.parcel-cache`
3. Relancez `npm install`
4. Relancez `npm start`

### Les modifications ne s'affichent pas?
1. Sauvegardez vos fichiers (Ctrl+S)
2. Le navigateur devrait se recharger automatiquement
3. Sinon, rafraîchissez manuellement (F5)

### Erreur de compilation?
1. Vérifiez la syntaxe JSX
2. Vérifiez les imports en haut des fichiers
3. Consultez la console pour voir l'erreur exacte
