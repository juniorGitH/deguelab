## 📊 FLUX DE DONNÉES

```
Utilisateur
    │
    ├─→ Clique sur "Acheter"
    │        │
    │        ▼
    │   JavaScript (handleBuy)
    │        │
    │        ├─ Récupère: product.name
    │        ├─ Récupère: product.price
    │        └─ Récupère: product.description
    │                │
    │                ▼
    │        Construit l'URL WhatsApp
    │        wa.me/22893733150?text=...
    │                │
    │                ▼
    └────→  window.open(whatsappUrl)
                    │
                    ▼
            Navigateur ouvre WhatsApp
                    │
                    ▼
            Application WhatsApp
                    │
                    ▼
            Degue Lab reçoit le message
```
