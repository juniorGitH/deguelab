## 📱 RESPONSIVE DESIGN

### Desktop (> 1024px):
- Navigation horizontale complète
- Grille de produits: 4 colonnes
- Sections côte à côte

### Tablet (768px - 1024px):
- Navigation avec tous les liens
- Grille de produits: 2 colonnes
- Sections empilées

### Mobile (< 768px):
- Menu hamburger (☰)
- Grille de produits: 1 colonne
- Tout empilé verticalement
- Boutons pleine largeur
